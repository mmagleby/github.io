1024-res Terrain Supplement for Hi-res Terrain Set

This is a supplement to the Hi-Res Terrain Set. It replaces a core set of 512x512 hi-res tiles with 1024x1024 tiles.  You will notice increased detail in the main fields, trenches, and airfield at low altitude when using these tiles.

This is an add-on to max188's Hi-Res Terrain Set for First Eagles, and requires the Hi-Res Terrain set already be installed.


INSTRUCTIONS

1) Make sure you have already installed max188's base Hi-Res Terrain set.

2) Back up your ..\ThirdWire\WWI\Terrain\wwiVerdun directory.

3 Unzip the contents of this zip file into the ..\ThirdWire\WWI\Terrain\wwiVerdun directory. Say "yes" to overwrite existing files. It
will replace 10 existing .bmp files with new ones.

4) That's it, you're done. Run the game and enjoy the terrain!


Notes:

1) If you want to go back to the original Hi-Res terrain just re-install the Hi-Res terrain set as per the original instructions.
Say "yes" to overwrite existing files. Or, just copy everything over from the backup you did in step 1.

2) You may notice a frame rate hit with these tiles if your video card does not have enough video RAM. I do not have any issues on my
system (256MB Video RAM).
