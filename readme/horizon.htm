EAW Horizon Sky -- README File

This horizon sky is based on Bird's excellent horizon clouds. The sky colors have been toned down a bit along with a couple other minor changes. This was originally intended to be used with the winter terrain, but looks very nice with the summer terrain as well. 


INSTRUCTIONS 

1) Unzip all the files to a directory.

2) Copy the two *.ter files to your EAW directory.



Notes:

1) If you use a terrain management program like STAB or Skins-n-More, you should copy the *.ter files to the appropriate directory instead of the EAW directory. Check the program's instructions on how to handle sky files.
