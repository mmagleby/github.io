Enhanced Terrain Set for Wings Over Europe

This terrain set is based on the Wings Over Europe default terrain tiles. It maintains the layout of the original terrain, and leverages many of the base textures. However, the overall color scheme, graininess, and shading of the original has been improved, along with many tweaks here and there. In addition, the tiles have been converted to 512x512 tiles, which improves some of the jagged edges and allows for some additional detail. The result is a terrain that still has a lot of the feel of the original terrain, but hopefully improves on it in many areas



INSTRUCTIONS

1) Back up your existing ..\Wings Over Europe\Terrain\GermanyCE directory.

2) Unzip the contents of this zip file into the existing ..\Wings Over Europe\Terrain\GermanyCE directory. Say yes if it asks to overwrite existing files.

3) That's it, you're done. Run the game and enjoy the terrain!



Notes:

1) If you want to go back to the default terrain, just delete the new files from the GermanyCE directory.

2) If you had some other terrain installed and want to go back to it, just copy everything from the GermanyCE backup directory to the current GermanyCE directory (you did back it up, right?). Or, just re-install the other terrain on top of this terrain.

3) Paint Shop Pro 8 was used to edit the tiles (except it does not handle targa file properly, had to switch back to ver 7 for this.)
