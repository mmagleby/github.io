Low Fuel Script -- README File

This script will modify the fuel burn rate values for player-flyable slots in a given planes.dat file. The rates are doubled from their original values, which seems to be more in line with actual fuel ranges in the game. You will need to be much more careful in managing your fuel with this mod. 

This affects both player and AI fuel burn rates, and results may be somewhat unpredictable. You may wish to experiment and fine tune these values accordingly. You can also change the values for the non-flyable plane slots by removing the REM statement for the given plane slot.  I have not validated any of these values, but they are based on what others have observed.


If you want to experiment with some of your own values, here is an example of how to edit the script.  Take for example the P51D section:

Jump( 1117 ) 'P51D
SetValue( TYPE_LONG, 150 ) 'Fuel Burn Rate x2


The value in the script of 150 is double the game's default burn rate. Thus, changing it to 75 will put it back at default as follows:

Jump( 1117 ) 'P51D
SetValue( TYPE_LONG, 75 ) 'Fuel Burn Rate x1 (the ' denotes a comment)


Likewise, if you want 150% burn rate, change the value to 75 x 1.5 = 113:

Jump( 1117 ) 'P51D
SetValue( TYPE_LONG, 113 ) 'Fuel Burn Rate x1.5


And so on...

INSTRUCTIONS:  This script is written to be run by ASHE. You can get the runtime version of this program from http://www.grandriversoftware.com/download/index.shtml Install ASHE RT and then open this script and run it. When you run the script, it will prompt for the location of the planes.dat file and make the cooresponding modifications. It can be run against any planes.dat file.
